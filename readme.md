# Character Counter

## Descripción
Este proyecto consiste en la maquetación de una interfaz web llamada "Character Counter", desarrollada utilizando HTML y CSS.

La aplicación permite visualizar un área de texto donde el usuario puede escribir contenido y observar diferentes métricas relacionadas con el texto ingresado, como:

- Cantidad total de caracteres.
- Cantidad de palabras.
- Cantidad de oraciones.
- Densidad de letras.
- Tiempo estimado de lectura.

## Tecnologías utilizadas

- HTML5
- CSS3

## Características

- Diseño responsive.
- Interfaz moderna inspirada en una herramienta de análisis de texto.
- Tarjetas estadísticas con información resumida.
- Barras de progreso para mostrar la densidad de letras.
- Uso de Flexbox y/o Grid para la distribución de elementos.

## Objetivos del proyecto

- Practicar la estructura semántica en HTML.
- Aplicar estilos avanzados con CSS.
- Trabajar con Flexbox y Grid.
- Replicar fielmente el diseño proporcionado mediante una imagen de referencia.
